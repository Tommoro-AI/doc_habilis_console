# 6.1 모델 추론

해당 탭에서는 학습된 모델 체크포인트를 이용해서 추론을 해볼 수 있습니다. 

1. 적절한 체크포인트를 선택한 후, 시작 버튼을 누르면 추론이 시작됩니다.

2. 추론을 중지하고 싶을 때는, 추론 중지 버튼을 누르면 추론이 중지 됩니다.
![예시 이미지](./image/inf_full2.png)
<div style="color:#2271b1">왼쪽 상단의 [포즈 불러오기] 기능을 이용해서 로봇의 초기 자세를 제어할 수 있습니다.</div>
<br />

## 체크 포인트 선택
Policy Paths에서 Choose policy 항목을 통해서 파일 탐색기를 열어볼 수 있습니다.

1. 파일 탐색기를 통해서 추론에 사용하고자 하는 체크포인트를 선택합니다.
2. 하빌리스 콘솔을 이용해서 학습한 체크포인트를 이용하여 추론하는 것을 추천합니다.

![예시 이미지](./image/inf_path.png)

<br />

> ⚠️ **주의**
> - 선택된 체크포인트가 유효하지 않은 경우 추론이 진행되지 않을 수 있습니다.
> - 하빌리스 콘솔에서 지원하지 않는 Policy
> - 하빌리스 콘솔에서 인식하지 못하는 metadata format
